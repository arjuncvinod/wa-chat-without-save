
## Chat without Save
## Installation

### Clone the Repository

```bash
git clone https://github.com/arjuncvinod/wa-chat-without-save.git
cd wa-chat-without-save
```
### Install dependencies
```console 
npm install
```
### Start the client
```console
npm start
```
```python
The Website will be available at http://localhost:3000
```
